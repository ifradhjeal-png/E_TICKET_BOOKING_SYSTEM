# E_TICKET_BOOKING_SYSTEM
.Shows Available Events: The program connects to a database and displays all events that have tickets left.

.Asks for User Input: It asks for your name, which event you want to see, and how many tickets you want to buy.

.Checks Ticket Count: It checks the database to make sure there are enough seats available for you.

.Calculates the Total Price: It finds the ticket price and multiplies it by the number of tickets you want.

.Confirms and Saves Booking: If you say "Yes" to confirm, it saves your booking details into a table and shows your unique Booking ID.

.Updates the Seats: Finally, it subtracts the tickets you bought from the total seats so the database stays accurate.
